# RTL Design and Verification of 4-bit ALU using Verilog HDL

# Overview
This project implements a 4-bit Arithmetic Logic Unit (ALU) using Verilog HDL. The design was functionally verified using a Verilog testbench and simulated using Xilinx ISE Simulator (ISim).

# Operations
- Addition
- Subtraction
- AND
- OR
- XOR
- XNOR
- Left Shift
- Right Shift

# Tools Used
- Verilog HDL
- Xilinx ISE 14.7
- ISim

# Files
- ALU_4bit.v
- ALU_4bit_tb.v
- Waveform Screenshot
